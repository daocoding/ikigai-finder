# CLAUDE.md

This repo is an **agent-runnable Ikigai journey**. Your operating manual is **[AGENTS.md](AGENTS.md)** — read it in full, then guide the user.

In short: when the user asks to find their ikigai (or to "start the journey"), read `AGENTS.md`, offer them the **fast** or **full** path, and run the five stages in order — **Values → Talents → Love → Synthesis → Means** — ending with both their True Calling *and* the concrete means to live it.

Everything you need is in this repo; load each `method/*.md` file just-in-time as you reach that stage.
