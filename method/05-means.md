# Stage 5 — Means → how to actually live it

**Goal:** turn the True Calling into **concrete ways to live it in the real world.** This is the half most "find your purpose" exercises skip — and the half the user actually needs to *do* anything.

> The calling is found inside you. **The means are found outside, in society.** A calling with no means is a daydream; means with no calling is a job you'll quietly hate. This stage marries them.

**This is the one stage where job titles, roles, companies, and businesses are exactly right.** Everything before now banned them; here they're the whole point. The same calling can be realized through many means — and that's freedom, not indecision: if one means closes, the calling carries into another.

You arrive holding the **True Calling**, and separately its **method**, **domain**, and **work purpose** (from Stage 4).

## Generate 3–5 means

Suggest a **range of forms**, not just full-time jobs — different levels of commitment so there's always a next step regardless of their situation:

- **Career paths / roles** (the obvious ones — name them plainly: "career coach," "UX researcher," "financial educator")
- **A side project or small business** that expresses the calling
- **A community or volunteer role**
- **A creative outlet** (writing, a channel, a workshop)

For **each** means, give the user all five of these — this structure is what makes it actionable rather than inspirational:

| Field | What it is |
|---|---|
| **Title** | A specific role or project name |
| **Day-to-day** | What this actually looks like on a Tuesday |
| **Why it fits** | How it expresses their **method + domain + purpose** (tie it back explicitly — this is the proof it's not a generic suggestion) |
| **First step this week** | One concrete action small enough to do in the next 7 days |
| **Timeframe** | short-term / medium-term / long-term |

Spread them across timeframes so there's both a "this weekend" option and a "two-year" option.

## Then: practices to live it *now*

Big paths take time; the calling shouldn't have to wait. Give them:

- **2–3 daily practices** — tiny habits that express the calling today, before any big leap. ("Spend 15 min explaining one money concept to a friend.")
- **1–2 weekly rituals** — a slightly bigger recurring practice that deepens alignment. ("Each Sunday, write one short post unpacking something you learned.")

These matter because they let the user *test the hypothesis* (Stage 4) immediately and cheaply — living the calling in miniature is how they find out if it fits.

## Reveal & visualize

Each row of the Means table is already a reveal — the **why it fits** column traces every suggestion back to method + domain + purpose, so nothing reads as generic advice. Optionally cap the stage with a simple **one-calling → many-means fan-out** diagram, so "many doors, one calling" is visible at a glance (if one door closes, the calling carries to the next). ## Close the journey

1. Fill in [`../templates/ikigai-result.md`](../templates/ikigai-result.md) with everything: pillars, work purpose, true calling, means, practices.
2. Hand it to the user as their keepsake.
3. End on **the single first step they committed to this week.** That's the only thing that turns this from a nice conversation into a changed life.
4. Offer to check in on how the first step went, and to re-run the journey later — the calling is a hypothesis, and hypotheses get refined by living.
