# Reference — 100 Values + Trap Values

Use this as a **controlled vocabulary** in Stage 1 ([`../method/01-values.md`](../method/01-values.md)): map the user's value-keywords onto these so the output stays consistent and method-aligned. It's an anchor, not a cage — if the user's truest word isn't here, keep their word. (Running in another language? Translate these as you go.)

## Trap values — probe these instead of accepting them

A trap value is a surface want that **depends on others' reactions** or is **just a means to something deeper**. When one shows up in the user's top values, don't take it at face value — ask the probe and help them name what's underneath.

| Trap value | Probe question | Likely deeper value |
|---|---|---|
| **Wealth** | "Money is a tool. Having it lets you feel — what?" | Freedom · Security · Comfort · Independence |
| **Abundance** | "What does abundance protect you from? Is it Peace of Mind?" | Peace of Mind · Security · Freedom |
| **Fame** | "Fame depends on others. If you were famous, what would that let you do?" | Impact · Connection · Expression · Recognition |
| **Popularity** | "Popularity is being liked. What do you get from being liked?" | Belonging · Safety · Connection · Acceptance |
| **Being Loved** | "'Being loved' is passive. What active value do you want to create?" | Connection · Loving · Intimacy · Belonging |
| **Victory** | "Victory is fleeting. What do you enjoy *during* the competition?" | Growth · Mastery · Challenge · Excellence |
| **Achievement** | "After you achieve the goal, how do you want to feel?" | Contribution · Self-Worth · Growth · Purpose |
| **Authority** | "We can't control others. What does authority let you exercise?" | Leadership · Responsibility · Impact · Service |
| **Correctness** | "Seeking 'correct' can be fear of being wrong. Is it Integrity or Truth?" | Integrity · Truth · Honesty · Excellence |

The pattern: **a value that depends on other people's reactions, or is only a means to something else, is a trap — dig one level down.**
